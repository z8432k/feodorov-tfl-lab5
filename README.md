#### Grammar

```ebnf {.line-numbers}
<S> ::= <A> | <D>
<A> ::= ab<B> | ac<B>
<B> ::= b<B> | ε
<D> ::= c<D> | b
```

##### Expanded for rule numbering

```ebnf
<B> ::= ε
<D> ::= c<D>

```
